In this page, you will be learning about what HTML and CSS are and how they are used. You will also learn about the uses of tags and attributes and be given examples of these.

HTML

HTML stands for Hypertext Markup Language. It is the standarized system for tagging text files to alter the font, color, graphics, and hyperlinks on Internet pages. Without HTML, there would be no websites or webpages. It is represented by the tag < html >.

CSS

CSS stands for Cascading Style Sheets. It is a stylesheet language used for describing the look and formatting of the HTML code. It saves a lot of work that originally be done in the HTML sheet. It is a separate sheet from HTML. CSS is mostly used when changing more than one element but with the same formatting.

Tags

Tags are keywords that are enclosed in angle brackets (used only in HTML). Always remember to close the tags or they will not work!

  • a tag- shown as < a >: inserts a hyperlink to another page
  • li tag- shown as < li >: defines a list item; it is used in ordered, unordered, and menu lists
  • div tag- shown as < div >: defines a section in the document
  • ol tag- shown as < ol >: creates a numbered list

Style tags are tags used in CSS and are used to describe and format the tags in HTML. They are keywords followed by curly brackets.

  • div { }: used to describe and format the words in the div tag in HTML

Attributes

An attribute is represented by a keyword, followed by an equal sign, and double quotes and are used to describe and format specific tags. It is used to customize tags in HTML format and is easiest to use when only changing one specific word or element.
