Welcome to the Computer Science Webpage! My name is Gabi Directo and I take the Computer Science class here at Summit Shasta. On this website, you will learn about the basics and advanced material of web designing. The main purpose of this class is to learn about the basics of the computer language and the science behind the websites we use everyday. Since technology is the base of our generation, it is important that we know how it works. Another important purpose of this class is to open your eyes to a whole new branch of jobs that require a computer science background.

Overview of Webpage

In this website we will be learning about the basics of coding/programming. This webpage will specifically cover the following items: (click on the picture for more information)



HTML-Hypertext Markup Language: the basic coding language used to create web pages.

CSS-Cascading Style Sheets: the stylesheet language used for formatting a document written in HTML.

Javascript: a scripting language used by clients of a browser; allows online user to alter the page.


AI: Artificial Intelligence- makes the webpage behave like a human being while still being completely computerized.

Robotics: a larger scale of computer science dealing with design, contruction, operation, and application of robots.

Game Design: the art of elaborating rules and mechanics to create interaction between computer and player or player and player for entertainment purposes.
